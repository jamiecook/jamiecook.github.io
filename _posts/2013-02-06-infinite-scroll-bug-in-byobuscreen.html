I was installing byobu on a precise pangolin vagrant box and found that when I launched it the screen started scrolling infinitely upwards.

Tracked it down to this bug report here https://bugs.launchpad.net/ubuntu/+source/byobu/+bug/952724

The solution is to edit your ~/.byobu/status file and change the tmux left status line from "logo" -> "#logo"
