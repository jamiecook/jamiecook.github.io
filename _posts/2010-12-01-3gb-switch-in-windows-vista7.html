Thanks to Volker for this little explanation of the changed process for enabling large memory space for user mode applications. Instead of adding /3GB to your boot.ini as you did inÂ windows XP, you now execute the following command from an administrator command prompt:

[sourcecode language="powershell" light="false"]
BCDEDIT.EXE /Set IncreaseUserVa 3072
[/sourcecode]
