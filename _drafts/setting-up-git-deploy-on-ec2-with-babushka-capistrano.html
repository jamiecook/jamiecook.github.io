Step 1
Boot the server and give it some initial config
* rvm installed
* add user for your application
* rvm and appropriate rubies installed for your application user
* babushka installed

Step 2

* make sure nginx is running
** sudo babushka benhoskings:'running.nginx'
** sudo babushka benhoskings:'postgres access'
**

babs benhoskings:'vhost enabled.nginx' path=~/current vhost_type=unicorn domain=...
babs benhoskings:'vhost configured.nginx' path=~/current vhost_type=unicorn listen_host=0.0.0.0 domain=...
babs benhoskings:'unicorn running' env=production
